Each automated update performs the following:

1. ‚úÖ Fetches fresh historical weather data (last 30 days by default)

3. ‚úÖ Combines historical and forecast data into a single dataset

## üìà Data Retention Policy

### Historical Data Window

**Default Period**: 30 days of historical data

**Configurable**: Yes, can be adjusted in `weather_forecast.py`

**Options Available**:
- 7 days (1 week)
- 14 days (2 weeks)
- 30 days (1 month, default)
- 60 days (2 months)

**Rolling Window**: Each update fetches the most recent N days, so the dataset maintains a consistent time window moving forward each day.

### Forecast Data

**Forecast Horizon**: 7 days ahead (168 hours)

**Update Frequency**: Daily (forecasts become more accurate as time approaches)

**Data Points**: ~168 hourly forecast points per update
